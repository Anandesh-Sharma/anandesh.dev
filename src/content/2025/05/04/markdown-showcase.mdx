# Markdown Showcase: Advanced MDX Features

This post demonstrates the full range of Markdown and MDX capabilities in our blog system. From basic formatting to complex code blocks with syntax highlighting, this guide covers it all.

## Basic Formatting

You can use **bold**, *italic*, or ***bold and italic*** text. You can also use ~~strikethrough~~ for text that's no longer relevant.

### Lists

Unordered lists:
- Item one
- Item two
- Item three
  - Nested item
  - Another nested item

Ordered lists:
1. First item
2. Second item
3. Third item
   1. Nested numbered item
   2. Another nested item

### Blockquotes

> Blockquotes are great for highlighting important information or quotes from others.
>
> You can have multiple paragraphs in a blockquote.
>
> > And even nested blockquotes!

## Code Blocks

### Inline Code

You can use `inline code` for short code snippets.

### Fenced Code Blocks


```python
def fibonacci(n):
    a, b = 0, 1
    for _ in range(n):
        a, b = b, a + b
    return a

print([fibonacci(n) for n in range(10)])
```

## React Components in MDX

MDX allows you to use React components directly in your markdown:

```jsx
import { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);
  
  return (
    <div>
      <p>You clicked {count} times</p>
      <button onClick={() => setCount(count + 1)}>
        Click me
      </button>
    </div>
  );
}
```
